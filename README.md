# 2 MW Offshore Wind Turbine — Design Project

Design of a 2 MW offshore wind turbine intended for Baltic Sea coastal conditions.
The project covers the full mechatronic design process, from aerodynamic blade
optimization to control system implementation and simulation.

The project includes blade design using the BEM algorithm and NREL S815 airfoil,
structural analysis of the drivetrain shaft, pitch mechanism, and tower using FEA
(ANSYS), selection of a DFIG generator with a back-to-back converter, design of
the full electrical and control system (Siemens S7-1500, SINAMICS), and
MATLAB/Simulink simulation of the complete turbine with pitch and yaw control.
